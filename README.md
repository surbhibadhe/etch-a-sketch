# Etch-a-Sketch

A browser-based drawing application inspired by the classic Etch-a-Sketch toy. Users can draw on a customizable grid, change colors, generate random colors, shade squares progressively, erase drawings, and adjust the grid resolution.

## Features

* Draw by clicking and dragging across the grid
* Adjustable grid resolution (1×1 to 100×100)
* Custom color picker
* Random rainbow color mode
* Progressive shading mode
* Eraser tool
* Toggle grid lines on/off
* Clear the entire sketch board

## Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/etch-a-sketch.git
```

2. Navigate to the project directory:

```bash
cd etch-a-sketch
```

3. Open `index.html` in your browser.
